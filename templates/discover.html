To start your hair care journey, use this page to figure out what hair type you have!
From wavy to curly to coily hair, we have information to figure out which the categories you fall into. Using this information, you can start styling your hair!

Wavy Hair Types

2A hair:

This hair type is very fine and thin with very slight “s” shape curves.

2B hair:

This type has more defined “s” shape curls, is more thicker, and tends to be more frizzier.

2C hair:

This hair type has the most “s” defined shapes and is the thickest of all the wavy hair types.


Curly Hair Types

3A hair:

This hair type has very defined curls and is very thick, but can also be a bit frizzy.

3B hair:

This hair type has tighter curls and has a combination of different curl textures.

3C hair:

This hair type has a lot of kinks along with very tight curls.


Coily Hair Types

4A hair:

This hair type is soft with very tightly coiled curls/kinks.

4B hair:

This hair type has very tight coils, but tends to be less soft and defined.

4C hair:

This hair type hair extremely tight curls, with a very voluminous style.
